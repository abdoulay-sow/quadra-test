# quadra-testsuite

## Set Local Env 
```
    cd quadra-testsuite
    ./env-test.sh
```
## Set Prod Env 
```
    cd quadra-testsuite
    ./env-prod.sh
```
## Deploy Env 
```
    cd quadra-testsuite
    ./env-prod.sh
    firebase deploy --only hosting,functions
```
